# Benchmark datasets

`smoke_testsrc` is generated by `scripts/prepare_assets.sh`. It is intended only
for installation checks and smoke runs.

Published benchmark runs must use `mot17_uadetrac_public`. Download MOT17 and
UA-DETRAC from their official distribution points:

- MOT17: <https://motchallenge.net/data/MOT17/>
- UA-DETRAC: <https://detrac-db.rit.albany.edu/>

Select the clips listed in `configs/datasets.yaml`, normalize each clip to
1920x1080 at 30 FPS, and place the resulting files under `data/benchmark/`.

Example normalization command:

```bash
ffmpeg -i INPUT -vf scale=1920:1080,fps=30 -c:v libx264 -preset slow -crf 18 OUTPUT
```

After preparation, replace each `SET_AFTER_PREPARATION` value in
`configs/datasets.yaml` with:

```bash
sha256sum data/benchmark/*.mp4
```

The runner rejects benchmark mode while a checksum is missing, uses a
placeholder, or does not match the local file.
